# Srebrna Sad - Polish Orchard Website

Simple website for Srebrna Sad apple orchard in Naruszewo, Poland.

## Project Structure

- **frontend/** - React + TypeScript (customer-facing website)
- **backend/** - Python FastAPI (API server)

## Features

### Phase 1 (Current)
- Information about the orchard
- Photo gallery
- Contact form

### Phase 2 (Next)
- Apple ordering system with inventory management
- Admin panel for apple types, prices, photos
- Order calendar and time slots

## Tech Stack
- **Frontend:** React + TypeScript
- **Backend:** Python FastAPI
- **Database:** MongoDB
- **Hosting:** GitHub (frontend), TBD (backend)
